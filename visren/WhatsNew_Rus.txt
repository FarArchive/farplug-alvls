Визуальное переименование файлов - плагин для ФАР Менеджера
===========================================================

Build 5 (27.10.2007)
--------------------
[!] Плагин требует ФАР билд 2275 или выше.
[+] Список шаблонов: доступна клавиша F1 и из открытого списка.
[+] Лист файлов: F3 - посмотреть длинное имя файла полностью
    (с легкой руки Вадима Егорова, плагин "ВС" :-)
[*] Откат переехал с клавиши F3 на F6.
[+] Лист файлов: клик правой клавишей мыши равнозначен F3.

Build 4 (13.10.2007)
--------------------
[!] Плагин требует ФАР билд 2272 или выше.
[!] Скомпилирован с обновленной библиотекой PCRE (версии 7.4),
    брать как обычно в подпапке ..\SRC\PCRE74.
[-] После перехода в/из (полно-)экранной консоли наблюдали
    неверные прорисовки диалога по F5.
    Аналогично и при "перетаскивании", пока ...
[*] ...запретил "перетаскивание" диалога (позже разберемся :-).
[-] Список шаблонов: наблюдали артефакты от комбобокса после F4.
[*] Список шаблонов: изменился формат [DM]/[TM]: YYYY.MM.DD/hh-mm-ss.
[+] С изменением АПИ ФАРа (2267 билд) стало возможно использовать
    Ins непосредственно в самом списке шаблонов НЕ ЗАКРЫВАЯ его.
[+] Список шаблонов: разделители слов можно изменять по F4
    непосредственно на шаблоне [T].
[+] Покажем кол-во файлов в сообщении "Сбор информации о файлах".
[+] Клавишей Esc можно прервать затянувшуюся загрузку файлов.
[+] Транслитерация имен файлов: русский <-> russkij.

Build 3 (17.08.2007)
--------------------
[+] Поддерживаются mp3-тэги 2-й версии.
    Если присутствует тэг 2-й версии используем его, иначе
    читаем тэг 1-й версии и переименовываем по нему.
[*] Процедура отката теперь идет с последнего файла к первому,
    для предотвращения конфликта имен.

Build 2 (09.08.2007)
--------------------
[!] Плагин требует ФАР билд 2251 или выше.
[-] Исправлен английский перевод в диалоге.
[-] Регэкспы: результат сопоставления всегда был равен true.
[+] Регэкспы: поддержка в строке замены ссылок на скобки,
    (так называемые подвыражения) - до 99 ссылок.
[*] Лист файлов: по Ctrl-Num5 прокрученные имена также
    возвращаются в положение по умолчанию, т.е. в начало,
    а то наблюдались баги с прорисовкой колонок.
[*] Лист файлов: по F5 позиция первого видимого элемента,
    автоматически подстраивается под положение курсора.
[*] Лист файлов: рисуем границу между колонками.
[*] Изменена логика работы - теперь снимаем выделение только
    с переименованных файлов.
[*] После выбора клавиши "Отмена" (при возникновении ошибки)
    теперь показываем инф-цию с кол-вом уже обработанных файлов.
[*] Теперь после отката отмечаем файлы, а также позиционируем
    курсор на первый файл.
[*] Изменена логика разбора строки масок - теперь маски
    [N] и [E] действуют в обоих полях ввода.
[+] Переименования файлов по содержимому mp3-тэга.
    Поддерживаются только (возможно пока) mp3-тэги 1-й версии.
[+] Добавил обработку клавиш в листе файлов: Ctrl-PgUp.
[+] Также действует и двойной клик левой клавишей мыши.
[-] Присутствовала ошибка при разборе маски с запятой - [N#,#].
[+] Теперь поддерживаются маски, отсчитанные с конца имени
    файла - [N-#-#] и т.д.
[-] Лист файлов: при сдвиге колонки и прокрутке имен, изредка
    наблюдали пропажу самого короткого имени.
